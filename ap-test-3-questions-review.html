• Example: nicotinic ACh receptor

2. Metabotropic receptors

• Slow

• Neurotransmitter binds → activates G-proteins/second messengers

• Changes cell activity more gradually

• Example: muscarinic ACh receptor

Easy summary:

Ionotropic = fast, direct ion channel.

Metabotropic = slow, uses G-proteins.
